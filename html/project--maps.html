Project

Maps

The Challenge

  • UXR perspective

  • Make pick up areas visible even on areas like greater London (Live screenshot Rentalcars.com)
  • Customer gave feedback that they do not see the relation between the map on the left and the cards on the right

  • Dev perspective

  • We cannot change the Google map design as far as we know

  • Business perspective

  • More BPD

My Role

  • Create a first story board together with the PO: "How can the map view be improved
  • Do technical research how to improve the Google map display in the way we wanted
  • Collaborate with developers about the technical opportunities
  • Create high end interactive visuals in Sketch to prove that a solution can be usable and delightful

The Design1/6

  • I found out, that Google offers a Styling Wizard for their map on mapstyle.withgoogle.com
  • I tested different variants using a Rentalcars.com brand colour scheme
  • The example shows an implementation of a map in Sketch I've styled using a JSON file
  • I created different themes using different JSON files

The Design2/6

  • Users gave feedback in surveys that they have difficulties to understand the splitted filter
  • This Sketch prototype shows that the car filter and the other filters can easily be combined

The Design3/6

  • This screen shows that the design offers the opportunities to tie map pins together with special announcements like "Best Price".

The Design4/6

  • This approach would also allow to show different kinds of locations like airports or train stations.

The Design5/6

  • Completely new would be the easy selection by region.

The Design6/6

  • An advanced car card design that would enable customers adding extras directly on the search results page.
